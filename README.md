## 📖 About the Project

This repository contains a structured set of Python programming exercises typically found in undergraduate lab coursework. Each experiment (`EXPT`) delves into a new concept and consists of multiple individual programs demonstrating the capability and syntax of Python.

It serves as a perfect reference for beginners learning Python, featuring progressive difficulty starting from raw input/output all the way to classes, inheritance, and advanced functions.

## 📂 Repository Structure

The experiments are divided logically into sequenced folders:

| Folder | Topic / Concepts Covered | Description |
| :--- | :--- | :--- |
| 📁 **[EXPT - 1](./EXPT%20-%201/)** | **Basics & I/O** | Introduction to Python, `print()` statements, reading user inputs and simple variables. |
| 📁 **[EXPT - 2](./EXPT%20-%202/)** | **Operators & Math** | Working with operators, performing arithmetic calculations (e.g., Simple Interest) and formula evaluation. |
| 📁 **[EXPT - 3](./EXPT%20-%203/)** | **Strings** | String manipulation, indexing, slicing, and checking for conditions like palindromes. |
| 📁 **[EXPT - 4](./EXPT%20-%204/)** | **Loops (While/For)** | Core looping constructs, basic iteration, and generation of simple mathematical series. |
| 📁 **[EXPT - 5](./EXPT%20-%205/)** | **Advanced Logic** | Further application of loops and conditional logic to solve problems like the Fibonacci series. |
| 📁 **[EXPT - 6](./EXPT%20-%206/)** | **Data Structures** | Introduction to fundamental collections such as Lists, Tuples, Dictionaries, and Sets. |
| 📁 **[EXPT - 7](./EXPT%20-%207/)** | **Arrays & Ranges** | Iterating over dynamic ranges and complex user-driven array processing tasks. |
| 📁 **[EXPT - 8](./EXPT%20-%208/)** | **Functions I** | Defining user-defined functions, handling positional and keyword arguments. |
| 📁 **[EXPT - 9](./EXPT%20-%209/)** | **Functions II** | Advanced function concepts, lambda functions, and modular code components. |
| 📁 **[EXPT - 10](./EXPT%20-%2010/)** | **OOP** | Object-Oriented Programming (Classes, Objects, Methods, Single and Multiple Inheritance). |
| 📁 **[Practice](./Practice/)** | **Additional Work** | Extra programming exercises and miscellaneous scripts used for revision and practice. |

## 🚀 How to Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ayushman2005-cmyk/PPML_EXPERIMENTS.git
   cd PPML_EXPERIMENTS
   ```

2. **Navigate into an experiment folder**:
   ```bash
   cd "EXPT - 1"
   ```

3. **Run any python file using the terminal**:
   ```bash
   python Prog1.py
   ```
